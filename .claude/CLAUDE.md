# Environment
This system is NixOS with the Niri window manager. Home manager is being used with NixOS experimental features turned on. This is how most of my programs are installed. Despite using NixOS, I use conventional dotfiles to configure many things, which live in ~/dotfiles. This is symlinked to my actual home directory with `stow .` from the dotfiles directory.

# General Coding
- Keep comments to a minimum, only comment when it helps clarify something complicated or verbose
- Comments should be in lowercase
- Variable names should be full words, not abbreviations, within reason
